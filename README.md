# Técnicas de Programação II - DSM Fatec Marília

Este repositório contém os estudos e implementações realizados durante o 3º semestre do curso de Desenvolvimento de Software Multiplataforma (DSM) na Fatec Marília.

## 📌 Sobre o Projeto
O foco desta disciplina é o aprofundamento em **Padrões de Projeto (Design Patterns)** utilizando Python, explorando como estruturar códigos de forma escalável e profissional.

## 🏗️ Conteúdo Estudado
Dividimos os padrões em três categorias principais:

* **Padrões Criacionais:** Focados em mecanismos de criação de objetos (ex: Singleton, Factory, Builder).
* **Padrões Estruturais:** Focados em como montar objetos e classes em estruturas maiores (ex: Adapter, Facade, Decorator).
* **Padrões Comportamentais:** Focados na comunicação e atribuição de responsabilidades entre objetos (ex: Observer, Strategy, Command).

## 🛠️ Tecnologias
* Python 3.x
* UML (Modelagem)
